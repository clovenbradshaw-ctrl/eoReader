Settings · LLM backend
backend
The narrow per-clause agent and the librarian commentary both use the same backend. Switch at any time.
Anthropic API key
Cloud, claude-haiku-4-5. Stored in this browser only. Best output quality; queries leave your device.
WebLLM · in-browser inference
Quantized model running in WebGPU. First load downloads the model (slow, cached afterward); subsequent calls are fast. No install, no telemetry, no cloud round-trip. Requires a browser with WebGPU support.
model
Ollama · local server
Talks to a locally-running Ollama instance. Run ollama serve and pull a model with ollama pull llama3.2 first. Higher quality than WebLLM, but requires install.
LM Studio · local OpenAI-compatible server
Talks to a locally-running LM Studio server (OpenAI-compatible). Load a model in LM Studio's UI and start its server. The model field can be left as "auto" to use whatever's loaded.
LLM call ceiling per ingest
A soft guardrail. When crossed, ingest auto-bumps the ceiling by 500 and logs to the console — it won't pause or interrupt. Set this very high (or low for hard stops on cloud backends).
